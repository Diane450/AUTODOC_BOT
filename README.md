**Telegram-бот для автоматической генерации документов по шаблонам.**  
Сервис избавляет от рутины: принимает шаблон DOCX и источник данных (Excel или Google Sheets), автоматически заполняет плейсхолдеры и возвращает готовые файлы.

---

## 🚀 Основные возможности

*   **Работа с шаблонами:** Поддержка `.docx` файлов с плейсхолдерами вида `{{PLACEHOLDER}}`.
*   **Гибкие источники данных:**
    *   **Excel (.xlsx):** Загрузка файла напрямую в чат.
    *   **Google Sheets:** Работа по ссылке (интеграция через Google API).
*   **Массовая генерация:** Создание сотен документов за несколько секунд на основе строк в таблице.
*   **Форматы выдачи:**
    *   **ZIP-архив:** Пакет отдельных документов (по одному на строку таблицы).
    *   **Merged DOCX:** Единый файл со всеми сгенерированными страницами (удобно для печати).

---

## 🛠 Технологический стек

*   **Язык:** Python 3.11
*   **Фреймворк:** [aiogram 3.x](https://docs.aiogram.dev/)

---

## Принцип работы

1.  **Загрузка шаблона:** Пользователь отправляет `.docx` файл, где переменные данные обозначены как `{{FIO}}`, `{{DATE}}` и т.д.
2.  **Загрузка данных:** Пользователь отправляет Excel-файл или ссылку на Google-таблицу. Заголовки столбцов должны совпадать с названиями плейсхолдеров в шаблоне.
3.  **Выбор режима:** Бот спрашивает, как вернуть результат (Архив или Единый файл).
4.  **Генерация:** Скрипт подставляет данные из каждой строки таблицы в шаблон и формирует итоговые файлы.
